# Project Lily – AI Dashboard Generator

InsightIq is a Python-based application that automatically generates an interactive dashboard from uploaded datasets.

Users can upload CSV or Excel files and instantly view:

• KPI metrics  
• Interactive charts  
• AI-style insights  
• Statistical summaries  

The system automatically analyzes the dataset structure and generates meaningful visualizations without manual configuration.

---

## Features

- Automatic dataset analysis
- Dynamic KPI generation
- Interactive charts (Plotly)
- Data insights generation
- Statistical summaries
- Modern dashboard UI

---

## Tech Stack

Python  
Streamlit  
Pandas  
NumPy  
Plotly  

---

## How to Run

1. Clone the repository

2. Navigate to the project - cd insightIq

3. Install dependencies  - run pip install -r requirements.txt

4. Run the app - streamlit run InsightIq.py
